# 메뉴 초기 데이터 추가

## 사전 조건

1. Docker 실행 중
2. PostgreSQL 컨테이너 실행 중
3. Spring Boot 서버(8080) 실행 중

## 방법 1: 스크립트 실행 (권장)

```bash
./docs/seed-menus.sh
```

실행하면 아래 메뉴가 자동 추가됩니다:

```
홈 (/)
학습 (/study)
  └ 프론트 엔드 (/study/frontend)
내 카드 (/my-cards)
설정 (/settings)
DnD Kit 실습 (/practice-dnd-kit)
  └ Level 1 - 기본 드래그 (/practice-dnd-kit/level1)
  └ Level 2 - 정렬 리스트 (/practice-dnd-kit/level2)
  └ Level 3 - 다중 컨테이너 (/practice-dnd-kit/level3)
  └ Level 4 - 그리드 정렬 (/practice-dnd-kit/level4)
  └ Level 5 - 트리 정렬 (/practice-dnd-kit/level5)
```

## 방법 2: 메뉴 관리 페이지에서 직접 추가

1. 프론트 서버 실행 (`npm run dev`)
2. http://localhost:5173/menu-manage 접속
3. 하단 "메뉴 추가" 버튼 또는 우클릭 → "하위 메뉴 추가"로 입력

## 방법 3: API 직접 호출

```bash
# 1차 메뉴 추가
curl -X POST http://localhost:8080/api/menus \
  -H "Content-Type: application/json" \
  -d '{"name":"메뉴이름","path":"/경로","sortOrder":1,"visible":true}'

# 하위 메뉴 추가 (parentId 지정)
curl -X POST http://localhost:8080/api/menus \
  -H "Content-Type: application/json" \
  -d '{"name":"하위메뉴","path":"/부모/하위","parentId":1,"sortOrder":1,"visible":true}'

# 전체 메뉴 확인
curl http://localhost:8080/api/menus/tree | python3 -m json.tool
```

## 참고

- `ddl-auto: update` 설정이므로 서버 재시작해도 기존 데이터 유지됩니다
- DB 초기화가 필요하면 Docker 볼륨을 삭제하세요:
  ```bash
  cd edu-flash-server
  docker compose down -v
  docker compose up -d
  ```
